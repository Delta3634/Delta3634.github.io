本文最后更新于 2025-05-14T11:59:51+08:00

前后端关系

一、前端

  • 看的见得页面（可视

  • 响应式——APP、PC端、小程序、桌面程序

  • 使用的语言：

    • 网页三件套：HTML、CSS、JavaScript

    • iOS APP：Object-C

    • Android APP： Java

    • 桌面程序：C++

二、后端

  • 给页面提供数据，与服务器打交道
  • 具体：
    • 处理业务逻辑
    • 处理文件
    • 处理数据库
    • 把处理好的结果返回给前端（之后前端将数据展示出来）
  • 语言：
    • Java
    • PHP
    • Python
    • Go
    • JavaScript

三、前后端关系

前后端分离，前端——静态界面，数据写死；后端——处理数据后通过接口返回给前端
