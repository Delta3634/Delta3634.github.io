一般文件的package名应与文件名一致，在别的文件中用到这个包中的方法（此方法要大写——相当于public）时需要导入这个包（带上go mod init时的路径）

注意：

  • import _"fmt"：给当前包起一个别名——匿名的——无法使用当前包的方法，但是会执行当前包内部的init()方法
  • import 别名"fmt"：给当前包起一个别名，别名.API()来直接调用
  • import ."fmt"：将当前包内的全部方法，导入到当前本包的作用中，当前包内的全部方法可以直接使用API来调用，不需要使用原包名.API来调用
